# Security Policy

## Reporting a Vulnerability

This is a static frontend application with no server-side code, user authentication, or sensitive data storage. However, if you discover a security concern (e.g., XSS vulnerability, supply chain risk in dependencies), please report it responsibly:

- Email: rafsunsheikh@gmail.com
- Or open a [GitHub Security Advisory](https://github.com/rafsunsheikh/The-Chronicle-of-Light/security/advisories/new)

Please do not open public issues for security vulnerabilities. I will acknowledge receipt within 48 hours and work on a fix promptly.
